# Dashboard

Projeto de um Dashboard para gerenciar tempo/recursos gastos com atividades de lazer/domesticas/educação. Desenvolvido para compor portfolio e implementar conceitos aprendidos.

---

### :computer: Stack

[![React][React.js]][React-url]  
[![MaterialUI][MaterialUI]][MaterialUI-url]   
[![express][express]][express-url]     
[![MongoDB][mongodb]][MongoDB-url]   
[![Node.js][node]][node-url]    

### :gear: Setup

```Bash
npm i
npm i --prefix client
npm i --prefix server
```

### :arrow_forward: Run

```Bash
npm run both
```
